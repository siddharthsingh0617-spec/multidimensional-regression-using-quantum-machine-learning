Multidimensional Regression using Quantum Machine Learning

This project explores multidimensional regression using Quantum Machine Learning (QML) techniques. Instead of relying solely on classical neural networks, the model uses parameterized quantum circuits (PQCs) and quantum feature encoding to learn complex nonlinear mappings between multiple input variables and continuous outputs.

The goal is to demonstrate how hybrid quantum–classical models can perform regression tasks and potentially offer advantages in feature representation, expressivity, and computational efficiency for high-dimensional problems.
